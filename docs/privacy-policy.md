# Sir Tab Privacy Policy

Effective date: April 10, 2026

Sir Tab is a Chrome extension that helps you view and manage tabs, tab groups,
and bookmarks from Chrome's side panel.

## Summary

Sir Tab does not sell user data, does not use advertising trackers, and does
not transmit your tab, bookmark, or browsing data to the developer's servers.
The extension processes data inside your browser to provide its features and
stores a small amount of preference data locally in Chrome storage.

## Information Sir Tab Accesses

Sir Tab may access the following categories of information:

- Open tabs in the current Chrome window
- Tab group metadata, such as group names, colours, and collapse state
- Bookmark and bookmark-folder data already stored in Chrome
- Extension preference data stored locally, such as selected theme and sidebar
  collapse state

## How Sir Tab Uses Information

Sir Tab uses Chrome permissions only to provide its user-facing features:

- `tabs`: to list tabs, activate tabs, create tabs, close tabs, and reorder tabs
- `tabGroups`: to display tab groups, create groups, update group names/colours,
  and move tabs between groups
- `bookmarks`: to display bookmarks, open bookmarks, create bookmarks or
  folders, and remove bookmarks or folders when requested by the user
- `sidePanel`: to display the extension UI in Chrome's side panel
- `storage`: to save local preferences such as theme choice and collapsed
  sidebar sections

## Local Storage

Sir Tab stores limited extension state in Chrome's local extension storage,
including:

- Selected sidebar theme
- Collapsed tab groups by window
- Collapsed bookmark folders

This data stays in your Chrome profile storage and is used only so the
extension can restore your preferences.

## Data Sharing

Sir Tab does not sell, rent, or share your personal information, browsing
activity, tabs, tab groups, or bookmarks with third parties.

Sir Tab does not use third-party analytics, ad tech, or remote code services.

## Remote Code and External Network Requests

Sir Tab does not download and execute remote code.

At the time of this policy's effective date, Sir Tab does not send tab,
bookmark, or preference data to an external server for product functionality.

## Data Retention

Preference data stored by the extension remains in your local Chrome profile
until:

- you change it in the extension,
- Chrome removes it,
- or you uninstall the extension.

## Your Choices

You can:

- stop using the extension at any time,
- remove saved extension data by uninstalling the extension,
- manage bookmarks and tabs directly in Chrome,
- and review extension permissions in Chrome's extensions settings.

## Children's Privacy

Sir Tab is not directed to children and does not knowingly collect personal
information from children.
